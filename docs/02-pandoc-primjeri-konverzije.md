# Pandoc

## Konverzija
```bash
Konverzija iz .md u .html format
pandoc MANUAL.txt -o example1.html
```
Ovo stavar web stranicu koju možeš pogledati

```bash
Konverzija iz .md u .docx fromat
README.md -o dokument.docx
```
Daje ti mogućnost da sada možeš pregledati i koristiti datoteku u Word-u

```bash
Konverzija iz .md u .pdf format
CHANGELOG.md -o primjer.pdf
```
Napravi dokument u ljepo spremljeno dokument koji možeš pročitati.
